All subsystems with methods and WSDL descriptions for instance "EE"

How is this report generated?


This report is generated by making getWsdl requests from RIA's Monitoring Security Server to all X-Road subsystems in the X-Road instance.

Subsystems with the Empty icon have no X-Road services available.

Subsystems with the Error icon either could not be reached by RIA's Monitoring Security Server or there was some other error during the request.
